Bookshelf

About the Project

This activity was created to allow students to look for new books to read as part of their class. You can click through the books, read summarys, and view illustrations if any.

Things I learned

  • How to create JavaScript objects
  • How to create dynamic HTML content with JavaScript
  • How to optimize images for online use

How It works

Use the arrows to view more books. Click on a book to view a summary and view illustrations.
