Abdominalpelvic Regions

About the Project

I created this game for a medical terminology course. Students had to memorized several terms and needed visual aids to remember the terms. This game works as interactive flash cards.

Things I learned

  • How to use draggables from jQuery UI
  • How to use droppables from jQuery UI
  • How to assign locations to droppables

How It works

To play the game, drag the terms and drop them in the correct location. If the box is incorrect, it will bounce back.
