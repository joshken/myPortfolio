Random Dance Activity

About the Project

This is an activity inspired by an activity that dance student do in class. They roll a die to specify what kind of dance they have to perform. This activity mimicks a die by randomizing the dance specifications.

Things I learned

  • How to animate objects with CSS
  • How to randomize separate objects
  • How to create a responsive design for mobile devices

How It works

To play the game, click each spec to "roll the die" and find out how to perform your dance.
