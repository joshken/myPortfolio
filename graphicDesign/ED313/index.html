About the Banner

I designed the banner for this course. The instructor wanted to include a lot of pictures in this banner so I designed the grid pattern to allow for more featured pictures.

About the Template

I continually developed and changed the structure of this template throughout my time working at BYU-Idaho. It was coded specifically to work with our Learning Management System (LMS) at the time, which was Brainhoney.

I had to find a lot of work-arounds in order to override Brainhoney's default styling. After I had resolved all of the styling issues, this template pattern became the standard for every course developed in Brainhoney.
