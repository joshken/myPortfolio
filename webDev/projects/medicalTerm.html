Medical Terminology Study Guide

About the Project

This was a very large project. Features were continually added throughout development. You combine prefixes, root words, and suffixes to create your own words and definitions.

Things I learned

  • How to filter arrays with JavaScript
  • How to sort arrays using JavaScript
  • How to use media queries to create print friendly documents

How It works

Select a prefix, root word, and suffix to combine a word. Click the book icon to see the definition.

The menu button reveals a list of words from each unit. Clicking on them will select them from the main page.
