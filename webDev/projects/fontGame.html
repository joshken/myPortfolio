Font Game

About the Project

This game was created for Typography students to practice identifying fonts for their exams.

Things I learned

  • How to randomize order using JavaScript
  • How to optimize design for mobile users
  • How to create accordion menus using JavaScript

How It works

To play the game, select a font name then a style. If you are correct, you earn points.
